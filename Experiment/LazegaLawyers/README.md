# Lazega Lawyers Dataset

This repository includes the **Lazega Lawyers dataset**, which is used in the experiments presented in the associated paper.

## Source

The dataset was obtained from:

https://www.stats.ox.ac.uk/~snijders/siena/Lazega_lawyers_data.htm

## Original Authors

Lazega, E. (2001). *The Collegial Phenomenon: The Social Mechanisms of Cooperation Among Peers in a Corporate Law Partnership.* Oxford University Press.

## Usage in This Repository

The dataset is included **solely for reproducibility purposes** of the experiments presented in this repository.

No modifications have been made to the original data unless explicitly stated in the code.
